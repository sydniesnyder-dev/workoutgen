Firebase Setup Instructions

  1. Go to Firebase Console
  2. Click "Add Project" and create a new project
  3. In your project, click the web icon (</>) to add a web app
  4. Register your app (name it "Workout Builder")
  5. Copy the firebaseConfig object from the code they show you
  6. Go to "Realtime Database" in the left menu and click "Create Database"
  7. Choose "Start in test mode" (we'll secure it later)
  8. Paste your Firebase config below and click "Save Config"
